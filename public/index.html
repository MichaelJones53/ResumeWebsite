Merge Sort

Merge sort is a means of sorting that recursively splits the data in half until it is all a single unit. Then, as it climbs back up the stack, it sorts and merges the data.
Merge sort is fairly tricky to implement but does provide a bit of an efficiency boost in its sorting time compared to easier to implement sorting algorithms.

Radix Sort

Radix sort is a unique and clever way of sorting numbers. It essentially sorts the number by its place values (tens, hundreds, thousands, etc.) and does it very fast for large datasets.
Creating the radix sort algorithm was an interestesting experience because of the uniqueness of the concept. I can appreciated the critical thinking involved, especially with regards to conceptualization and implementation.

AVL Tree

An AVL tree is a self-balancing binary tree. This essentially means that from any given node, everything to the left is of a lower value, and everthing to the right is of a greater value, and the height of the left tree and right tree will differ by no more than one level.
Creating the AVL tree was a very challenging project. Keeping track of the tree height and performing proper insertion and rotation were key in implementing this data structure.
